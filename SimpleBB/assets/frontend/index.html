Password
 
Please, ensure that your name and password meet all the requirements.

Requirement 1. Your name must be unique in the system.

Requirement 2. Minimum password length is 16 symbols.

Requirement 3. Password length must be a multiple of four, e.g. 16, 20, 24, 28, 32, ...

Requirement 4. Password must contain only following allowed symbols:
  • Numbers from 0 to 9,
  • Capital latin letters from A to Z,
  • Special symbols:
    • ' ' (White Space),
    • '!' (Exclamation mark),
    • '"' (Quotation mark),
    • '#' (Number sign),
    • '$' (Dollar sign),
    • '%' (Percent sign),
    • '&' (Ampersand),
    • ''' (Apostrophe),
    • '(' (Left parenthesis),
    • ')' (Right parenthesis),
    • '*' (Asterisk),
    • '+' (Plus sign),
    • ',' (Comma),
    • '-' (Hyphen-minus),
    • '.' (Full stop or period),
    • '/' (Solidus or Slash),
    • ':' (Colon),
    • ';' (Semicolon),
    • '<' (Less-than sign),
    • '=' (Equal sign),
    • '>' (Greater-than sign),
    • '?' (Question mark),
    • '@' (At sign or Commercial at),
    • '[' (Left Square Bracket),
    • '\' (Backslash),
    • ']' (Right Square Bracket),
    • '^' (Circumflex accent),
    • '_' (Low line).
